# Aprende Trading - Portal de Educación Financiera

![Trading Guide](https://images.unsplash.com/photo-1611974717411-957bb7270d42?q=80&w=1200&auto=format&fit=crop)

Este proyecto es un portal integral diseñado para guiar a traders principiantes desde cero hasta su primera operativa real. Combina recursos educativos, herramientas técnicas y acceso a una comunidad vibrante.

## 🚀 Características Principales

- **Guía de Inicio Rápido:** Roadmap de 4 pasos para configurar cuentas y plataformas.
- **Ecosistema Técnico:** Selección curada de las mejores herramientas (XM, MT5, TradingView, Binance).
- **Centro Comunitario:** Información sobre nuestra comunidad en Discord y agenda de sesiones en vivo.
- **Formación Avanzada:** Enlace directo a la Academia Slange para aquellos que buscan profesionalismo.
- **Optimización Mobile-First:** Experiencia fluida en cualquier dispositivo con navegación adaptativa y flexbox grids.
- **SEO Ready:** Etiquetas Meta (Open Graph, Twitter Cards) por página utilizando `react-helmet-async`.
- **Organización Centralizada de Enlaces:** Componente Constants con todos los enlaces de marketing y afiliación (Binance, XM Demo/Real).

## 🛠️ Stack Tecnológico

- **Framework:** [React 18+](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Routing:** [React Router 6](https://reactrouter.com/)

## 📦 Instalación y Desarrollo

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/usuario/aprende-trading.git
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción:**
   ```bash
   npm run build
   ```

## 🌐 Despliegue

La aplicación está lista para ser desplegada en plataformas como Vercel, Netlify o Cloud Run. Simplemente conecta tu repositorio y configura el comando de build a `npm run build` y el directorio de salida a `dist`.

## 📄 Descargo de Responsabilidad

El trading en mercados financieros implica un riesgo significativo de pérdida de capital. El contenido de este portal es estrictamente educativo y no constituye asesoría financiera profesional.
